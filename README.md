# How to obtain the result
Run file "run.py" with all True or False set to True to generate all simulations and to save all the data.
Check so the names on the file is correct for save and no saved data will have the name "no name".

#### Method
To get the figures of the method, run the two seperated python files in the folder "plot_method".

#### Number of obstacles
To get the figures and data of the number of obstacles, run the file "plota.py" in the folder "number_obs".

#### Design choice of obstacles
To get the figures and data of the design choice of obstacles, run the file "plota_d.py" in the folder "design_c".

#### Standard devision in desired speed
To get the figures and data of the standard devision in desired speed, run the file "plotaWW.py" in the folder "variance_of_speed".
